# permutations
simple permutations example for school
## About
This program will take a string and print all possible permutations. For example the string "ABC" will result in:
<br>
ABC
<br>
ACB
<br>
CAB
<br>
CBA
<br>
BCA
<br>
BAC
<br>
## Note:
For a string of length n, there exists n! permutations.
